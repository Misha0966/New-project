 Применение СЧС в истории


- N = 1812  # год войны
- m = 2     # разбиваем на 2 части
- k = 7     # коэффициент преобразования

**разбиение**→ ["18", "12"]

**уможение по частям** → ["126", "84"]**построчное объединение в число** → PQ = "12684"

**сравнием**

N * k = 1812 × 7 = "12684"

PQ = 12684

Полное совпадение!
