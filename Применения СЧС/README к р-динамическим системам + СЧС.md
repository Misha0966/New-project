# Документация к коду: `р-динамическая система + СЧС.jl`

**Structural Numerical Symmetry (SNS) in p-adic systems**  

## Автор: Ющенко Михаил Юрьевич


## Назначение:

[Программа](https://github.com/Misha0966/New-project/blob/main/%D0%9F%D1%80%D0%B8%D0%BC%D0%B5%D0%BD%D0%B5%D0%BD%D0%B8%D1%8F%20%D0%A1%D0%A7%D0%A1/%D1%80-%D0%B4%D0%B8%D0%BD%D0%B0%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F%20%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0%20%2B%20%D0%A1%D0%A7%D0%A1.jl) проводит вычислительный эксперимент по выявлению **Структурной Числовой Симметрии (СЧС)** в **p-адической системе счисления**.  

СЧС определяется как совпадение (полное или частичное) между:
- **Глобальным результатом**: \( \text{NK} = (N \cdot k) \bmod p^L \)
- **Локальным результатом**: \( \text{PQ} = \text{concat}\big( (N_i \cdot k) \bmod p^{|N_i|} \big) \),  
где \( N_i \) — части p-ичной записи числа \( N \), разбитые на `m` сегментов.

Эксперимент подтверждает, что СЧС — **не артефакт десятичной системы**, 
а **фундаментальное свойство позиционных числовых структур**, проявляющееся даже при жёсткой декомпозиции.


## Параметры:

| Параметр     | Тип     | Описание |
|-------------|---------|--------|
| `START_N`   | `Int`   | Начало диапазона чисел \( N \) |
| `STOP_N`    | `Int`   | Конец диапазона (включительно) |
| `M`         | `Int`   | Количество частей при разбиении числа (`m ≥ 2`) |
| `K`         | `Int`   | Множитель для локального и глобального умножения (`k ≥ 2`) |
| `P`         | `Int`   | Основание p-адической системы (**должно быть простым**) |
| `L`         | `Int`   | Длина p-адического приближения (модуль \( p^L \)) |


**Рекомендации**:  
- Для видео: `STOP_N - START_N + 1 ≤ 5000`  
- Для больших диапазонов: отключите визуализацию (закомментируйте вызов `plot_sns_frame`)


## Выходные файлы:

| Файл | Формат | Содержание |
|------|--------|-----------|
| `results_sns_p_adic.csv` | CSV | Полные данные по каждому \( N \): разбиение, PQ, NK, тип совпадения |
| `statistics_sns_p_adic.txt` | TXT (UTF-8) | Сводная статистика на русском языке |
| `sns_p{P}_m{M}_k{K}_N{START}-{STOP}.mp4` | MP4 (H.264) | Видео динамики СЧС (только при N ≤ 5000) |
| `frames/` | PNG | Временные кадры (удаляются вручную) |


## Типы совпадений (в данных):

| Метка | Условие |
|------|--------|
| `ПОЛНОЕ СОВПАДЕНИЕ` | `PQ == NK` |
| `СОВПАДАЮТ НАЧАЛО И КОНЕЦ` | Совпадают ≥1 символ слева **и** ≥1 символ справа |
| `СОВПАДАЕТ ТОЛЬКО НАЧАЛО` | Совпадают символы слева, но **не** справа |
| `СОВПАДАЕТ ТОЛЬКО КОНЕЦ` | Совпадают символы справа, но **не** слева |
| `НЕТ СОВПАДЕНИЙ` | Ни одно совпадение не обнаружено |


## Архитектура:

1. **`base_p_string`** — преобразует \( N \) в строку длины `L` по основанию `p`.
2. **`split_string_equal`** — разбивает строку на `m` частей (максимально равной длины).
3. **`multiply_preserve_length_p`** — умножает часть, сохраняя её длину по модулю \( p^{\text{len}} \).
4. **`compare_pq_nk`** — классифицирует совпадение без статистики, только по структуре.
5. **`plot_sns_frame`** — генерирует кадр видео с визуализацией СЧС.
6. **`run_sns_p_adic_video`** — основной цикл анализа с многопоточной статистикой.
