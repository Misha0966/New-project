# Класс В- совпадение границ,  ( при переносе разрядов в середине числа) т.е. старших и младших разрядах между:
# PQ ( результатом цифровой композиции результатов умножения блоков исходного числа, а не арифметической суммы. ) и Nk- классическим умножением

**Есть 2 типа доказательства:**

-1. Локальное т.е. доказательство наличия границ ( не составляет особого труда, при должном уровне смекалки )

-2. Глобальное ( полное доказательство ) : объяснение не только совпадения границ, но так же объяснения, почему в одних случаях наблюдается "зеркальные границы", т.е.
начальная и конечная цифра полностью совпадают, а в других нет, то есть имеют различия? Но и так же поведение самой середины числа? Хаотична
ли она, если да, то почему? Есть хоть какая то связь между поведением середины с самими границами? Как она соотносится с "зеркальными" и с
"не зеркальными" границами результата PQ? И согласуется ли они с серединой? Если нет - то почему? Если да - почему?
